Known issues
------------
- This has not been tested on Windows 11.

Manual Installation
-------------------
(1) Choose a directory for AutoArrange, and (2) copy AutoArrange.exe and AutoArrange.exe.confg into that directory.

(3) Optional: Add a shortcut to AutoArrange to your Windows Startup folder ("%AppData%\Microsoft\Windows\Start Menu\Programs\Startup").

(4) Optional: Ensure your Taskbar Notification area makes AutoArrange visible at all times. It is useful to be able to quickly stop AutoArrange and manually move windows around.
